# FlightController-Tests

Neste repositório encontram-se alguns trechos de códigos relevantes feitos para validar um sistema embarcado para controle 
de voo de quadricópteros desenvolvido para a obtenção do diploma de mestrado. Para sua execução, é necessário utilizar as bibliotecas
citadas ao longo da dissertação e que estão disponíveis em outros repositórios Git também referenciados.
